# LumoChat

LumoChat - учебный мобильный мессенджер на Flutter для курсовой работы по разработке приложений.

## Что реализовано

- регистрация и вход пользователя;
- редактирование профиля: имя, аватар и краткая информация;
- поиск пользователей по имени или email;
- список диалогов;
- личная переписка между двумя пользователями;
- групповые чаты;
- отправка текстовых сообщений;
- отправка изображений в чат;
- отображение автора и времени отправки сообщения;
- статус прочтения сообщений;
- светлая и темная темы оформления;
- локальное сохранение пользователей, диалогов и сообщений;
- индикаторы загрузки и сообщения об ошибках.

## Технологии

- Flutter;
- Dart;
- SharedPreferences для локального хранения данных демонстрационного MVP;
- Image Picker для выбора аватаров и изображений в чатах;
- Crypto для хеширования паролей.

Пароли не хранятся открытым текстом. В приложении сохраняется SHA-256 хеш пароля.

## Как запустить

Перед запуском нужен установленный Flutter SDK.

Проверить установку:

```bash
flutter doctor
```

Установить зависимости:

```bash
flutter pub get
```

Запустить приложение:

```bash
flutter run
```

Если на Windows нет подключенного Android-устройства, можно запустить desktop-версию:

```bash
flutter run -d windows
```

Для запуска на Android нужно:

- установить Android Studio;
- установить Android SDK;
- включить отладку по USB на телефоне или запустить Android Emulator;
- проверить устройство командой:

```bash
flutter devices
```

После этого:

```bash
flutter run
```

## Проверочный сценарий

1. Создать первый аккаунт.
2. Выйти из аккаунта.
3. Создать второй аккаунт.
4. Найти первого пользователя через поиск.
5. Открыть личный диалог и отправить сообщение.
6. Выйти и войти под первым аккаунтом.
7. Убедиться, что сообщение сохранилось.
8. Ответить во второй аккаунт.
9. Проверить отправку изображения в чат.
10. Создать групповой чат.
11. Проверить статус прочтения сообщений.
12. Переключить светлую и темную тему.

## Примечания

Приложение сделано как MVP для учебной проверки. Данные сохраняются локально на устройстве, поэтому переписка остается доступной после повторного входа в приложение на том же устройстве.
