# LumoChat

LumoChat - учебный мобильный мессенджер на Flutter с backend на Firebase.

## Что реализовано

- регистрация и вход через Firebase Authentication;
- редактирование профиля: имя, аватар и краткая информация;
- хранение пользователей, диалогов и сообщений в Cloud Firestore;
- хранение аватаров и изображений сообщений в Cloud Firestore;
- сохранение FCM-токена пользователя для push-уведомлений;
- поиск пользователей по имени или email;
- список диалогов;
- личная переписка между двумя пользователями;
- групповые чаты;
- отправка текстовых сообщений;
- отправка изображений в чат;
- отображение автора и времени отправки сообщения;
- статус прочтения сообщений;
- светлая и темная темы оформления;
- индикаторы загрузки и сообщения об ошибках.

## Технологии

- Flutter;
- Dart;
- Firebase Authentication;
- Cloud Firestore;
- Firebase Cloud Messaging;
- Image Picker;
- SharedPreferences только для сохранения выбранной темы оформления.

Пароли не хранятся в приложении. За регистрацию, вход и безопасное хранение учетных данных отвечает Firebase Authentication.

## Подключение Firebase

Проект уже подключен к Firebase и содержит готовые файлы конфигурации:

- `lib/firebase_options.dart`
- `android/app/google-services.json`
- `firestore.rules`
- `firebase.json`

Дополнительно настраивать Firebase для обычного запуска не нужно.

## Правила Firestore

В проект добавлены файлы:

```text
firestore.rules
firebase.json
```

Их можно загрузить в Firebase командой:

```bash
firebase deploy --only firestore:rules
```

## Как запустить

Проверить окружение:

```bash
flutter doctor
```

Установить зависимости:

```bash
flutter pub get
```

Запустить приложение:

```bash
flutter run
```

Для запуска на Windows:

```bash
flutter run -d windows
```

Для запуска на Android нужно подключить телефон с USB debugging или запустить Android Emulator.

Проверить устройства:

```bash
flutter devices
```

## Проверочный сценарий

1. Создать первый аккаунт.
2. Выйти из аккаунта.
3. Создать второй аккаунт.
4. Найти первого пользователя через поиск.
5. Открыть личный диалог и отправить сообщение.
6. Выйти и войти под первым аккаунтом.
7. Убедиться, что сообщение сохранено в Firestore и отображается после повторного входа.
8. Ответить во второй аккаунт.
9. Проверить отправку изображения в чат через Cloud Firestore.
10. Создать групповой чат.
11. Проверить статус прочтения сообщений.
12. Переключить светлую и темную тему.

## Примечание

Изображения сообщений и аватары сохраняются в Cloud Firestore в сжатом виде, поэтому проект работает на Firebase Spark plan без привязки банковской карты и без Firebase Storage.
